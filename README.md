# Overview

This is a Flappy Bird-inspired game that I created as part of my journey to learn the Arcade library in Python. This is my very first game, and I chose to develop a Flappy Bird-like experience because I found it fun, simple, yet addictive. In my version, the game is called "Splashy Fish." Instead of a bird, I created a fish that swims through coral obstacles. The goal of the game is to keep the fish from hitting the ground, the ceiling, or the coral obstacles, and to achieve the highest score possible. You play the game by pressing the spacebar, which causes the fish to jump. The longer you avoid obstacles, the higher your score climbs.

[Video Demonstration of Splashy Fish](https://youtu.be/x-i-_rOcf3E?si=Y1h6XEhMukNsKK7b)

# Development Environment

For the development of this game, I used the Python programming language, specifically version 3.12. The game was built using the Arcade library. I developed the game in Visual Studio Code, which served as my primary code editor. Additionally, I utilized standard Python libraries such as math for mathematical operations, os for file handling, and random for generating random values.
